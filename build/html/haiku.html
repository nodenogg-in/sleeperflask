haiku script¶

haiku.count_syllables(word)¶

Function to count syllables in a word using the CMU dictionary. If word is not found in dictionary, use syllable_count_for_unknown_word.

haiku.generate_line_from_extracts(extracts, num_syllables)¶

Function to generate prompt for user input

haiku.generate_line_from_lm(prompt, model, tokenizer, extracts, num_syllables)¶

Function to generate text based on a given prompt

haiku.get_words_for_first_n_syllables(text, n)¶

Function to extract words adding up to a specific number of syllables from the beginning of a text.

haiku.get_words_for_last_n_syllables(text, n)¶

Function to extract words adding up to a specific number of syllables from the end of a text.

haiku.main()¶

Main function to load data, models and generate output based on user inputs.

haiku.random_split(sentence)¶

Function to split a sentence at a random index.

haiku.syllable_count_oov(word)¶

Function to estimate syllable count for a word not found in CMU dictionary. This is a simple rule-based approach that won’t be perfect.

haiku.trim(s1, s2)¶

Removes any overlap from the beginning of the second string.
