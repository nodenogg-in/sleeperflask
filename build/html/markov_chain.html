markov_chain module¶

This script uses Markov Chains to generate text based on a set of input text files.

RUN:

pip3 install -r requirements.txt python markov_chain.py <corpus_path> <notes_path> <length> [–context_size <context_window>]

param corpus_path

Path to the readinglist folder containing the corpus.

type corpus_path

str

param notes_path

Path to the notes folder containing the notes.

type notes_path

str

param length

Length of text to be generated.

type length

int

param –context_size

Optional. Size of the context window for the Markov chain. Defaults to 1.

type –context_size

int

Example

python markov_chain.py corpus_folder/ notes_folder/ 100 –context_size 2
